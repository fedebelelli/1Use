¿Qué es OneUse?

OneUse es una plataforma web que nos permite sacar provecho de aquellos objetos que no usamos y tenemos en nuestras casas ociosamente, alquilandolo a otras personas que precisen de dicho objeto mediante una transaccion segura y confiable! A través de OneUse, no sólo podrás ahorrar el dinero al no comprar, sino que podrás sacar partido a tus objetos, evitando así el consumo excesivo y los desplazamientos innecesarios

Beneficios de nuestra plataforma

Angular

Angular, es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.

Angular Material

Angular Material es una librería de componentes web con un diseño Material design, una guía de estilo creada por Google para Android y para sus aplicaciones.

ExpressJS

Express es una infraestructura de aplicaciones web Node.js mínima y flexible que proporciona un conjunto sólido de características para las aplicaciones web y móviles.

MongoDB

MongoDB es un sistema de base de datos NoSQL orientado a documentos de código abierto. En lugar de guardar los datos en tablas, MongoDB guarda estructuras de datos BSON con un esquema dinámico, haciendo que la integración de los datos en ciertas aplicaciones sea más fácil y rápida.

NodeJS

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor basado en el lenguaje de programación ECMAScript, asíncrono, con E/S de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google.
